Book Cover

Online Cover Fetching

When local extraction fails (no embedded art, sidecar image, or archive cover), search Open Library and Google Books for a cover image. No API key required. Covers are cached by Jellyfin after first fetch.

Rendering

Resolution for PDF first-page rendering. Default: 150. Higher = better quality but slower. 72 for fast, 300 for high quality.
Output compression level (1-100). Default: 85. Lower values produce smaller files.
Maximum time for pdftoppm or ffmpeg to process a single file. Default: 30.

Library Status

Libraries with Book Cover enabled as an image fetcher. To enable it for a library, go to Dashboard → Libraries → [Library] → Image Fetchers.
