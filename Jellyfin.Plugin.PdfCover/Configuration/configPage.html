Book Cover Settings

Fallback cover provider for books and audiobooks. Extracts PDF first pages via pdftoppm, searches EPUB archives for cover images, and extracts embedded artwork from audio files (MP3, M4A, M4B, FLAC, etc.) via ffmpeg — handling mislabeled codec tags that break Jellyfin's built-in Image Extractor.

Online Cover Fetching

When local cover extraction fails (no embedded art, no sidecar image), search Open Library and Google Books for a cover image.

Sources: Open Library (openlibrary.org) → Google Books (books.google.com). No API key required. Covers are cached by Jellyfin after first fetch.
